# STRIX v2 - Executive Intelligence System
## AI 기반 3단계 보고 업무 자동화 시스템

### 🎯 핵심 가치
**"보고 준비 시간 90% 단축, 인사이트 품질 200% 향상"**

### 🚀 Quick Start (3분 설정)

1. **Excel 파일 생성**
   - `STRIX_Executive_v2.xlsm` 생성
   - 매크로 사용 통합 문서로 저장

2. **VBA 모듈 설치**
   ```
   Alt + F11 → 모듈 추가 → 코드 붙여넣기
   ```

3. **대시보드 생성**
   ```
   Alt + F8 → CreateExecutiveDashboard → 실행
   ```
